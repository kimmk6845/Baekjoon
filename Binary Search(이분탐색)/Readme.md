# 이분탐색(Binary Search)
선형 자료 구조에서 대표적인 탐색 방법 중 하나로 정렬된 배열에서 탐색 범위를 절반씩 줄여가며 찾고자 하는 값을 찾는 방법

이미 정렬되어 있는 배열에만 사용할 수 있음

시간 복잡도가 O(N)인 선형탐색보다 효율적인 탐색 방법을 진행하므로 O(logN)의 시간 복잡도를 갖음
